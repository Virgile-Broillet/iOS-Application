# Guess the Number Game (iOS)

This is a fun iOS app where the goal is to guess a number randomly generated by the app. Every game is different, and you can play as many times as you want!

---

## Features

- Random number generation to guess.
- Simple and intuitive interface to enter your guess.
- Immediate feedback if your guess is too high, too low, or correct.
- Option to play again after each round.
- Compatible with iOS devices.

---

## How to Play

1. Launch the app on your iOS device.
2. Enter a number in the input field.
3. The app will tell you if your guess is too low, too high, or correct.
4. If you guess correctly, you can start a new game.
